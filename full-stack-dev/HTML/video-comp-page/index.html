Run Less, Run Faster

This webpage provides tools describing the 3 key runs as part of the Run Less, Run Faster approach to distance training.

Track Intervals

Track Intervals, or Repeats, improve VO2 max, running speed, and running economy. As the name suggests, the intention is to repeat certain distances (intervals) with a target pace in mind.

Tempo Run

Tempo Runs improve endurance by raising the lactate threshold. They typically are run over longer distances than track intervals at a slower pace than intervals, though still considered 'hard'.

Long Run

Long Runs improve endurance, in contrast to Tempo Runs, by raising aerobic metabolism. They tend to be longer mileage runs than Tempo Runs at a more 'pedestrian' pace, considered about 30 seconds per mile slower than goal race pace.
